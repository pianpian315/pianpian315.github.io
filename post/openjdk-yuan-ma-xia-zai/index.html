OpenJDK源码下载

2019-09-29

一、源码下载

1、安装mercurial版本控制

$ brew install mercurial

mercurial安装完毕后，可以通过hg命令来下载源码，使用方式和git、svn类似。

2、下载OpenJDK源码

$ hg clone http://hg.openjdk.java.net/jdk8/jdk8 jdk8

你可以看到clone下来的代码很少，那是因为OpenJDK将不同类型的source文件放在了不同的子仓库，你可以进入OpenJDK的目录，执行get_source.sh文件即可。

$ cd jdk8
$ chmod +x get_source.sh
$ ./get_source.sh

3、配置JDK（可选）

$ bash configure

4、构建编译

$ make clean images

或者

make clean images LOG=debug // to display DEBUG information

你也可以单独对某个模块进行编译构建

$ make jdk
$ make hotspot
$ make test

二、源码基本说明

— corba：不流行的多语言、分布式通讯接口
— hotspot：Java 虚拟机
— jaxp：XML 处理
— jaxws：一组 XML web services 的 Java API
— langtools：Java 语言工具
— nashorn：JVM 上的 JavaScript 运行时
— jdk：java 开发工具包
jdk是我们研究源码经常会用到的，里面包含：
— 针对操作系统的部分
— share：与平台无关的实现
